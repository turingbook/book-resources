## Python及依赖库

运行源码需要安装以下库：

* NumPy
* Matplotlib
* PyTorch（版本：2.x）
* torchvision
* tqdm

※Python版本请使用**3.x**系列。

## 运行方法

进入各章节对应的文件夹，直接用Python命令运行即可。

```
$ cd step01
$ python norm_dist.py

$ cd ../step02
$ python generate.py
```
